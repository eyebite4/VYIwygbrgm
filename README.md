# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行岩石检测
- 提供9类岩石识别：玄武岩、闪长岩、花岗岩、大理石、石英岩、白垩、煤、石灰岩、砂岩
- 支持选择不同的检测模型
- 界面实时显示目标位置、目标总数和置信度信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图

## 适用场景
- 地质勘探与研究
- 矿产资源评估
- 岩石样本分析
- 地质教学与研究
- 环境监测
- 建筑材料检测
- 考古研究

## 技术特点
- 基于YOLOv8深度学习框架
- 使用PyQt5构建用户界面
- 包含4300张标注数据集
- 提供完整源码和预训练模型
- 支持Python 3.9环境
- 集成OpenCV进行图像处理
- 包含详细的项目运行教程
